🏔️ SnowPeak Business Card - Print Ready

Professional business cards with your actual logo, designed to match your website's elegant aesthetic.

📏 Specifications:
  • Size: 3.5" × 2" (Standard US Business Card)
  • Resolution: Print Quality (Vector SVG)
  • Design: Gradient front with logo, clean back
  • Colors: Exact brand colors (#395ECA → #848BD8)

🖨️ How to Print (IMPORTANT - Read Carefully!):

  1. Print to PDF with Background Graphics:
    • Press Ctrl+P (Windows) or Cmd+P (Mac)
    • ⚠️ CRITICAL: Click "More settings" or "Show more settings"
    • ✅ CHECK "Background graphics" - This ensures the gradient background prints!
    • Select "Save as PDF" as the printer
    • Choose "Letter" paper size
    • Set margins to 0.5 inches
    • Set scale to 100%
    • Save the PDF
  2. Send to Professional Printer:
    • Use the PDF file created above
    • Recommended: VistaPrint, Moo, or local print shop
    • Specify: 3.5" × 2" business cards
    • Paper: Premium cardstock (16pt or higher)
    • Finish: Matte or glossy (your preference)

💡 Printing Tips:

  • Paper Quality: Use thick cardstock (16pt-18pt) for professional feel
  • Finish Options: Matte for elegant, Glossy for vibrant colors
  • Quantity: Order 250-500 cards (better price per card)
  • Rounded Corners: Optional upgrade for modern look

⚠️ Important Notes:

To view the logos properly:

Open this file by running a local server (e.g., npm run dev) and navigating to http://localhost:3000/business-card.html

The logos use relative paths and need to be served from a web server to display correctly.

🚨 For Best PDF Quality:

  • Chrome/Edge: Enable "Background graphics" in print settings
  • Firefox: Check "Print backgrounds" in print settings
  • Safari: Check "Print backgrounds" in the print dialog
  • If background still doesn't print, try a different browser (Chrome works best)
  • The CSS now forces backgrounds to print, but browser settings can override this

📋 Multiple Cards Per Page

When printing, you can fit multiple cards on one page for cutting. Most professional printers will handle this automatically, but if printing at home:

  • Print on 8.5" × 11" cardstock
  • You can fit 10 cards per page (2 columns × 5 rows)
  • Use a paper cutter for clean edges
  • Leave small margins between cards for cutting
